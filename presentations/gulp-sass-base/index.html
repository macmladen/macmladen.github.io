gulp-sass-base

The boilerplate for modern
front end workflow

Created by MacMladen / @MacMladen

Hi everyone!

The problem we are looking to solve with this boilerplate is the shift from ruby based workflow to node based one.

Also, along with switching to a faster way to produce CSS we will also try to standardize, introduce structure and many more modern workflow that takes much more than just makin' CSS.

The problem with ruby

It's not you — it's me

Really, the problem is not in ruby as such, it is excellent language.

It is also not just a shift for shifting

It is a much wider quest.

Current ruby stack

  • ruby system
  • rvm (ruby version manager)
  • gems
  • bundler
  • Gemfile
  • config.rb

$ bundler install
$ compass watch

Problems

  • System ruby usually outdated
  • Ruby version 1.8.3, 1.9.3, 2.2.2
  • Different gemsets / dependencies
  • Multiple gem installation paths
  • Gems/ruby complexities
  • Rarely anyone knows ruby
  • Setting in local
  • Setting on server
